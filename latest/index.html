MLMetrics.jl's Documentation

This package represents a community effort to provide a wide range of common performance metrics for evaluating learning algorithms and prediction models. As such it is part of the JuliaML ecosystem.

Where to begin?

If this is the first time you consider using MLMetrics for your machine learning related experiments or packages, make sure to check out the "Getting Started" section; specifically "How to …?", which lists some of most common scenarios and links to the appropriate places that should guide you on how to approach these scenarios using the functionality provided by this or other packages.
